## 一、必须删除的填充短语（开场白和拐杖词）

| 禁用表达 | 替代方向 |
|---------|---------|
| 首先…其次…最后… | 自然过渡，或不用序号 |
| 总之/综上所述 | 直接写结论 |
| 值得注意的是 | 直接陈述 |
| 不可否认的是 | 直接陈述 |
| 毫无疑问 | 删掉，或用更温柔的表达 |
| 一般来说 | 酌情保留或改为"多数时候" |
| 在当今快速变化的时代 | 删掉这类冗余开场白 |
| 随着社会的不断发展 | 删掉 |
| 应该注意的是 | 删掉 |
| 更重要的是 | 删掉，直接说 |
| 不仅如此 | 删掉或改写 |

## 二、必须打破的公式结构

### 2.1 三段式排比
```
❌ 首先，A。其次，B。最后，C。
✅ A，再说B，至于C——其实也没那么简单。
```

### 2.2 对比句式固化
```
❌ 与其A，不如B。
❌ 不是A，不是B，而是C。
✅ 用具体场景/故事替代抽象对比
```

### 2.3 破折号戏剧性揭示
```
❌ 这背后，是一个让人意外的真相——它其实……
✅ 直接说出来，不卖关子
```

### 2.4 公式化结尾
```
❌ 希望你……
❌ 愿每一个……的你……
❌ 记住，……
✅ 开放式提问、温柔收束、场景收尾
```

### 2.5 冗余段落骨架
```
❌ 每段都是：观点 + 例子 + 小结
✅ 结构有变化：故事 → 感受 → 疑问 → 场景 → ...
```

## 三、AI 高频词汇黑名单

以下词汇在AI文本中出现频率异常高，使用时需警惕：

**职场/成功类**：赋能、赋予、颠覆、破局、升维、降维打击、底层逻辑、顶层设计、
闭环、迭代、内化、系统性、结构性、深度剖析、全面分析

**情感/成长类**：蜕变、涅槃、升华、内耗、疗愈（过度使用）、拥抱变化、
解锁、探索、践行、落地、抓手

**表达类**：毫无疑问、不言而喻、显而易见、众所周知、不可或缺、
举足轻重、至关重要

## 四、语态与节奏规则

### 4.1 主动语态优先
```
❌ 这个问题被很多人忽视了
✅ 很多人忽视了这个问题
```

### 4.2 句子长度变化
- 避免连续3句以上长度相同的句子
- 偶尔用极短句（1-5字）打节奏：
```
✅ 我试了很久。没用。
```

### 4.3 段落结尾多样化
- 不能每段都以"总结性陈述"结尾
- 可以是问句、场景描述、留白、转折

## 五、增加人味的方法

### 5.1 有观点，不只是报告
```
❌ 这种现象在年轻人中较为普遍，原因是多方面的。
✅ 我觉得这不只是懒，是某种说不清楚的绝望感。
```

### 5.2 承认不确定性
```
❌ 这个方法非常有效，可以帮助你提升效率。
✅ 对我来说有点用，不保证对每个人都管用。
```

### 5.3 具体细节 > 抽象总结
```
❌ 那段时间很辛苦，但我坚持下来了。
✅ 那段时间每天6点起，挤地铁，盯着Excel，回家倒头就睡。
```

### 5.4 删除"金句"
```
❌ 生活从不辜负努力奔跑的人。
✅ 说不定累了就好了（更像真人说的话）
```

## 六、快速自查清单

修改完成后，对照以下清单检查：

- [ ] 没有"首先/其次/最后"
- [ ] 没有"值得注意的是/更重要的是"
- [ ] 没有"毫无疑问/不可否认"
- [ ] 没有公式化结尾（希望/愿你/记住）
- [ ] 没有连续3段相同句式结构
- [ ] 没有过多破折号做戏剧性揭示
- [ ] 有短句打节奏（不全是长句）
- [ ] 结尾不是"总结式陈述"
- [ ] 包含至少一处具体细节
- [ ] 语气与账号文风档案一致
