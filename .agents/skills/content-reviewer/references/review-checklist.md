# 内容审查清单

## 快速审查（5分钟内）

### AI 痕迹快查
- [ ] 开头是否有冗余套话？
- [ ] 是否有"首先/其次/最后"三段式？
- [ ] 结尾是否是公式化（希望/愿你/记住）？
- [ ] 是否有过多AI高频词（赋能、深度剖析等）？
- [ ] 破折号是否用于"戏剧性揭示"超过2次？

### 文风一致性快查
- [ ] 人称是否与账号风格一致？
- [ ] 整体语气基调是否符合？
- [ ] 段落长度是否符合账号偏好？
- [ ] 开头方式是否符合账号常用模式？
- [ ] 结尾方式是否符合账号常用模式？

---

## 深度审查（完整版）

### 一、词汇层
- [ ] 使用了账号 writing-style.md 中的高频特色词汇？
- [ ] 是否出现了 writing-style.md 禁忌清单中的禁用词？
- [ ] 情绪词密度是否符合账号风格？
- [ ] 是否有AI高频词汇（对照 anti-ai-patterns.md 黑名单）？

### 二、句式层
- [ ] 短/中/长句比例是否符合账号节奏？
- [ ] 段落长度是否在账号偏好范围内？
- [ ] 开头模式是否符合（场景/提问/直述）？
- [ ] 结尾模式是否符合？
- [ ] 段落之间的过渡是否自然？

### 三、语气层
- [ ] 主要人称与账号一致？
- [ ] 语气基调（温柔/犀利/幽默等）符合账号气质？
- [ ] 口语化程度合适？
- [ ] 情绪密度（高/中/低）符合账号风格？

### 四、结构层
- [ ] 文章骨架是否符合账号典型结构？
- [ ] 段落数是否在合理范围内？
- [ ] 小标题使用频率是否符合账号偏好？
- [ ] 列表使用是否适度？

### 五、内容层
- [ ] 主题是否符合账号栏目定位？
- [ ] 观点是否有个人立场（不只是中立报告）？
- [ ] 是否有具体细节而非抽象总结？
- [ ] 字数是否在账号偏好范围内？

---

## 偏离度评分标准

| 分数 | 含义 |
|------|------|
| 9-10 | 高度符合文风，几乎感觉不到AI痕迹 |
| 7-8 | 基本符合，有1-3处小问题 |
| 5-6 | 一般，有明显的AI痕迹或风格偏离 |
| 3-4 | 较差，结构公式化，语气不符 |
| 1-2 | 严重不符，需大幅改写 |
